# Dyslexia Text Simulator

A Unity + TextMeshPro prototype that simulates unstable text movement for a dyslexia-like reading experience.

本项目是一个基于 Unity 和 TextMeshPro 的阅读体验模拟原型。  
它通过逐字符控制文字的位置、旋转、移动和回归，使普通读者体验一种文字不稳定、难以聚焦的阅读状态。

## Project Status

This project is an unfinished prototype.

目前项目仍是半成品，代码结构和运动逻辑还不完善，但已经可以展示核心概念。  
短期内我暂时不会继续重构，因此先开源出来，供有兴趣的人参考、修改或继续实现。

## Features

- 基于 TextMeshPro 的逐字符顶点控制
- 根据注视点附近区域触发文字运动
- 支持文字移动、旋转、缩放、离开后回归
- 可用于阅读障碍体验模拟、交互文字实验、视觉可访问性教学展示

## Important Note

This project is not a medical tool.

本项目不能代表所有阅读障碍者的真实体验，也不能用于诊断、治疗或医学评估。  
它只是一个视觉交互模拟，用来帮助普通人初步感受某种文字不稳定造成的阅读困难。

Reading difficulties and dyslexia are complex. They are not simply “moving letters”.  
This project only simulates a visual disturbance effect and should not be treated as a complete or accurate medical simulation.
